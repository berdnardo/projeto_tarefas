# Projeto Tarefas


### Projeto utilizado para desenvolver os conhecimentos em backend e frontend


O conceito do projeto foi desenvolver uma plataforma onde um professor/personal possa gerenciar os exercicios executados pelos seus alunos, obtendo o controle de quais atividades, de quando e do intervalo em que foram realizados, como tambem a lista de todos os alunos sobre sua assistÃªncia.


## Principais Tecnologias Utilizadas

#### Frontend:

* VueJS
* Axios
* Vue Router
* Vuex
* Vuetify

#### Backend:

* TypeScript
* PostGreSQL
* Express
* TypeORM
